Projetos

Análise e classificação de imóveis para compra e venda

O contexto do projeto trata de responder duas questões feitas pela equipe de negócios de uma empresa imobiliária fictícia chamada House Rocket Company: 1) Quais são os imóveis que a House Rocket deveria comprar e por qual preço? 2) Uma vez comprados os imóveis, qual o melhor momento para vendê-los e por qual preço? Para tais respostas, foi coletada e analisada uma base de dados no Kaggle.

Ferramentas utilizadas:

  • Python 3.0.
  • Jupyter Notebook.
  • PyCharm.
  • Git e Github.
  • Streamlit Cloud.
